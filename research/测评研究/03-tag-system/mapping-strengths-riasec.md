# CliftonStrengths Themes to RIASEC / TAG Dimensions Mapping

> **重要声明 / Important Statement**: 本映射表为**理论假设性映射（theoretical hypothetical mapping）**，用于整合研究设计，未经过实证验证。每个主题的映射均基于其 Gallup 官方定义与行为描述，而非基于统计相关。

---

## 1. 映射逻辑 / Mapping Logic

### 1.1 映射维度
对每个 CliftonStrengths 主题，给出以下映射：

- **Primary RIASEC**：最可能对应的 Holland 类型；
- **Secondary RIASEC**：次要对应类型；
- **Ability Tags (ABIL-)**：相关的 TAG 能力标签；
- **Value Tags (VAL-)**：相关的 TAG 价值标签；
- **Potential Tension (TENS-)**：该主题在职业情境中可能带来的张力。

### 1.2 映射原则
1. 基于 Gallup 官方主题定义（Gallup, 2017）\[D]；
2. 结合主题行为描述与 RIASEC 类型定义进行判断；
3. 一个主题可映射到多个 RIASEC 类型，但需区分主次；
4. 所有映射标注证据等级为 [C]（理论推断）。

## 7. 应用说明 / Application Notes

1. **主类型优先**：在初步匹配职业集群时，优先考虑主题的 Primary RIASEC。
2. **次类型补充**：当学生兴趣剖面中出现相邻或混合类型时，Secondary RIASEC 可提供补充解释。
3. **领域分布分析**：统计学生 Top 5/10 主题在四大 CliftonStrengths 领域的分布，可与 RIASEC 剖面进行对比。
4. **张力识别**：利用 `TENS-` 列识别学生可能在职业情境中遇到的挑战，用于职业对话。

## 8. 批判性反思 / Critical Reflection

1. **映射的主观性**：本映射基于对主题定义的理论解读，不同研究者可能给出不同映射。
2. **文化适用性**：CliftonStrengths 主题在美国组织文化中开发，其在中文语境中的含义可能有所不同。
3. **未验证性**：本映射未经过统计验证（如因子分析、相关分析或结构方程模型），不可作为效度证据。
4. **避免决定论**：主题映射仅提示亲和方向，不意味着个体“只能”从事某类职业。
