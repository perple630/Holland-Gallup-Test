# RIASEC × CliftonStrengths 双维度测评落地框架
# Practical Assessment Framework: Holland RIASEC × Gallup CliftonStrengths

> **版本 / Version**: 1.0  
> **定位 / Positioning**: 本框架面向学校、培训机构与个体使用者，提供一套可落地、可复制的测评-解读-建议流程。框架强调 **Holland RIASEC 与 Gallup CliftonStrengths 分开施测、分别产出 Tags、再通过矩阵整合**，并同时提供 **学生可直接使用的简化版** 与 **测评师使用的专业版**。  
> **性质 / Nature**: 理论整合与实践工具，未经过实证验证，仅用于自我探索与职业对话。

---

## 1. 框架核心理念 / Core Ideas

### 1.1 两个测评分别回答不同问题

| 测评工具 | 核心问题 | 测评结果 | 对应的 Tag 类型 |
|---------|---------|---------|----------------|
| **Holland RIASEC** | “我对什么类型的工作/学习内容感兴趣？” | 三字母代码（如 IAS、SEC、RIC） | `RIASEC-` 兴趣标签 |
| **Gallup CliftonStrengths** | “我倾向于以什么方式思考、行动与建立关系？” | Top 5 / Top 10 天赋主题 | `STR-` 优势标签 |

### 1.2 为什么先分开，再整合？

- **避免混淆**：兴趣 ≠ 优势。一个人可能对艺术（A）很感兴趣，但未必具备 Ideation 或 Futuristic 等创造性优势；反之亦然。
- **分别验证**：学生先分别理解两个结果，再观察它们之间的一致、补充与张力。
- **分层使用**：学生版只需关注“我的兴趣方向”和“我的优势风格”；专业版才进入 Tag 匹配、亲和度估算与张力分析。

### 1.3 双维度整合公式

> **有效职业方向 ≈ 我喜欢的舞台（RIASEC） × 我擅长的表演方式（CliftonStrengths） × 现实条件**

## 2. 第一步：Holland RIASEC 测评与 Tag 产出

### 2.1 施测说明

- 使用工具：SDS 自评量表、O*NET Interest Profiler、Truity Holland Code Test 等。
- 输出结果：六个维度得分 + 三字母代码。
- 提醒学生：RIASEC 测的是“兴趣倾向”，不是能力或性格。

### 2.2 RIASEC Tag 生成规则

| 结果层级 | 生成的 Tag | 说明 |
|---------|-----------|------|
| 主导类型 | `RIASEC-<第一字母>` | 如 `RIASEC-I` |
| 三字母代码 | `RIASEC-<第一>-<第二>-<第三>` | 如 `RIASEC-I-A-S` |
| 相邻/相对提示 | `HEX-adjacent:<类型1>-<类型2>` 或 `HEX-opposite:<类型1>-<类型2>` | 用于后续张力分析 |

### 2.3 学生可直接理解的兴趣标签（学生版）

将六个类型转化为“工作舞台”隐喻：

| RIASEC 类型 | 学生版 Tag | 舞台比喻 |
|------------|-----------|---------|
| R 现实型 | `#动手实践派` | 车间、工地、实验室操作台 |
| I 研究型 | `#探索思考派` | 图书馆、实验室、数据分析室 |
| A 艺术型 | `#创意表达派` | 画室、舞台、写作空间 |
| S 社会型 | `#助人连接派` | 教室、诊所、咨询室 |
| E 企业型 | `#影响领导派` | 会议室、商场、创业空间 |
| C 常规型 | `#秩序组织派` | 办公室、数据中心、档案室 |

### 2.4 测评师版补充：剖面特征指标

| 指标 | 含义 | 专业解读提示 |
|-----|------|-------------|
| **区分度（Differentiation）** | 六类型得分差异是否明显 | 高区分度：兴趣轮廓清晰；低区分度：需更多探索 |
| **一致性（Consistency）** | 三码在六边形上是否相邻 | 高一致性（如 I-A、S-E）：职业方向较聚焦；低一致性（如 I-E、A-C）：存在内在张力 |
| **剖面高度（Profile Elevation）** | 整体得分是否偏高/偏低 | 高分：兴趣广泛或作答倾向肯定；低分：需排除缺乏自我了解 |

## 3. 第二步：Gallup CliftonStrengths 测评与 Tag 产出

### 3.1 施测说明

- 使用工具：Gallup CliftonStrengths（Top 5 / Top 10 / 完整 34）。
- 输出结果：34 项主题排序 + 四大领域分布。
- 提醒学生：CliftonStrengths 测的是“自然才能/优势表达风格”，不是能力高低。

### 3.2 CliftonStrengths Tag 生成规则

| 结果层级 | 生成的 Tag | 说明 |
|---------|-----------|------|
| Top 5 主题 | `STR-<theme>` | 如 `STR-learner`、`STR-empathy` |
| 主导领域 | `DOMAIN-<领域名>` | 如 `DOMAIN-relationship-building` |
| 领域分布 | `DOMAIN-MIX:<领域1>=<数量>,<领域2>=<数量>` | 用于可视化 |

### 3.3 学生可直接理解的优势标签（学生版）

将四大领域转化为“工作风格”：

| CliftonStrengths 领域 | 学生版 Tag | 风格比喻 |
|----------------------|-----------|---------|
| Strategic Thinking 战略思维 | `#动脑策划型` | 喜欢研究、分析、规划、想新点子 |
| Executing 执行力 | `#实干完成型` | 喜欢按计划把事情做完、负责到底 |
| Influencing 影响力 | #表达推动型 | 喜欢说服、领导、带动他人行动 |
| Relationship Building 关系建立 | `#连接陪伴型` | 喜欢理解他人、建立深层关系、凝聚团队 |

### 3.4 测评师版补充：主题组合分析

| 分析维度 | 专业解读提示 |
|---------|-------------|
| 领域集中度 | Top 5 是否集中在 1-2 个领域？高度集中：优势风格鲜明；分散：能力表达多元 |
| 第一主题能量 | 学生的第一主题往往是其最自然、最稳定的能量来源 |
| 主题组合叙事 | 不孤立解释单个主题，而是构建“主题组合故事” |
| 领域短板提示 | 不视为缺陷，而是提醒学生在哪些情境下可能需要策略性补充 |

|:---|:---|:---|:---|:---|
| **R 现实型** | 技术研发工程师、数据工程师、法医技师、技术研究员 | 项目经理、施工主管、质量控制员、设备运维经理 | 技术销售、产品演示专家、创业技术合伙人 | 技术培训师、设备维护顾问、运动教练 |

| **A 艺术型** | 概念设计师、艺术研究员、品牌策略师 | 出版编辑、展览执行、舞台监督 | 创意总监、品牌公关、艺术经纪人 | 艺术治疗师、创意写作教师、社区艺术组织者 |

| **E 企业型** | 管理顾问、投资分析师、商业战略家 | 运营总监、项目经理、企业高管 | 创业者、销售总监、投资人、政治家 | 客户关系总监、HRD、组织发展顾问 |
| **C 常规型** | 财务分析师、精算师、数据架构师 | 会计师、审计师、行政经理、运营专员 | 销售运营经理、采购经理、商务拓展 | 客户服务经理、HR 运营、员工关系专员 |

### 4.2 学生版矩阵解读卡

为每个交叉象限设计一句“你可以这样理解自己”：

| RIASEC \ 优势领域 | 战略思维 | 执行力 | 影响力 | 关系建立 |
|:---|:---|:---|:---|:---|
| **现实型** | 你喜欢用头脑解决技术难题 | 你喜欢亲自动手把项目完成 | 你喜欢把技术价值讲给别人听 | 你喜欢手把手教别人操作 |
| **研究型** | 你享受追根究底的思考过程 | 你擅长把研究变成可执行流程 | 你乐于把复杂知识讲清楚 | 你喜欢陪伴他人学习与成长 |
| **艺术型** | 你总在寻找新创意和新视角 | 你能把创意一步步做出来 | 你善于展示和推广自己的作品 | 你通过艺术与他人建立连接 |
| **社会型** | 你喜欢分析人的需求并设计帮助方案 | 你踏实可靠，能把服务做到位 | 你善于激励和带领群体 | 你天生擅长倾听和理解他人 |
| **企业型** | 你擅长看清商业机会和趋势 | 你目标明确，能带队拿到结果 | 你享受说服、谈判和竞争 | 你通过关系网推动事情发生 |
| **常规型** | 你擅长从数据中发现规律 | 你注重细节，能把流程做到极致 | 你能在规则内推动合作 | 你是团队稳定可靠的后盾 |

## 5. 第四步：学生直接使用版 / Student-Friendly Version

### 5.1 报告结构（一页纸版）

#### 我的兴趣舞台（RIASEC）

```
我的三字母代码：__________

我的主导舞台：□ 动手实践派（R）  □ 探索思考派（I）  □ 创意表达派（A）
              □ 助人连接派（S）  □ 影响领导派（E）  □ 秩序组织派（C）

这意味着我喜欢的工作环境是：_____________________________
```

#### 我的优势风格（CliftonStrengths）

```
我的 Top 5 主题：
1. ____________  2. ____________  3. ____________
4. ____________  5. ____________

我的主导风格：□ 动脑策划型  □ 实干完成型  □ 表达推动型  □ 连接陪伴型

这意味着我最自然的工作方式是：_____________________________
```

#### 我的“兴趣 × 优势”交叉点

```
从上方矩阵中找到你的交叉象限：

____________________________________________________________

3 个我可以优先了解的方向：
1. ________________
2. ________________
3. ________________
```

#### 我的下一步探索行动

```
□ 找 1 位从事该方向的人聊天
□ 参加 1 次相关体验课/工作坊/志愿活动
□ 读 1 本相关入门书或看 1 个纪录片
□ 在选课/选专业时优先尝试相关领域
□ 记录 1 周“什么时候我最投入、最有成就感”
```

### 5.2 学生版建议示例

**示例 A：RIASEC = SAE，CliftonStrengths 主导 = Relationship Building**

| 维度 | 结果 | 学生版解读 |
|-----|------|-----------|
| 兴趣舞台 | S（助人）+ A（创意）+ E（影响） | 你喜欢在“帮助别人”的舞台上，用有创意、有影响力的方式工作 |
| 优势风格 | Relationship Building | 你 naturally 擅长理解他人、建立信任、陪伴成长 |
| 交叉方向 | 艺术治疗师、教育培训机构创始人、员工福祉负责人、企业学习体验设计师 | 你可以优先考虑“用温暖关系 + 创意方法 + 社会影响”帮助他人 |
| 探索行动 | 体验一次艺术疗愈工作坊；采访一位教育创业者 | 用真实体验验证兴趣和优势 |

**示例 B：RIASEC = IRC，CliftonStrengths 主导 = Strategic Thinking**

| 维度 | 结果 | 学生版解读 |
|-----|------|-----------|
| 兴趣舞台 | I（研究）+ R（现实）+ C（常规） | 你喜欢有逻辑、有结构、能落地的技术研究 |
| 优势风格 | Strategic Thinking | 你 naturally 喜欢分析、学习、找规律 |
| 交叉方向 | 数据科学家、生物医学研究员、精算师、质量工程师 | 你可以优先考虑“用深度思考解决实际问题”的方向 |
| 探索行动 | 尝试一个小数据分析项目；参观大学实验室 | 验证你是否喜欢“研究 + 实操”的组合 |

### 5.3 学生版“避坑提醒”

1. **兴趣不是能力**：测试结果说你对某方向感兴趣，不代表你已经会了，还需要学习。
2. **优势不是喜欢**：你擅长的事不一定是你最爱做的事，要结合兴趣一起看。
3. **不要被标签困住**：测试结果只是当前状态的快照，你可以成长和变化。
4. **多去体验和聊天**：真正做决策前，尽量实习、参观、采访从业者。
5. **听取多方意见**：家长、老师、行业前辈的建议都可以参考，但最终决定权在你。

## 6. 第五步：测评师专业版 / Practitioner Version

### 6.1 专业版解读流程

```
步骤 1：分别审核两个测评结果的质量
        ↓
步骤 2：产出 RIASEC Tags 与 CliftonStrengths Tags
        ↓
步骤 3：使用 4×6 矩阵定位候选职业集群
        ↓
步骤 4：使用 TAG 数据库计算兴趣-优势亲和度
        ↓
步骤 5：识别可能性、张力与发展建议
        ↓
步骤 6：生成双版本报告（学生版 + 专业版）
        ↓
步骤 7：进行 1 对 1 或团体解读对话
```

### 6.2 专业版 Tag 整合表

| 学生信息 | Tag 产出 | 专业解读要点 |
|---------|---------|-------------|
| RIASEC 三码 | `RIASEC-X-Y-Z` | 主导类型、一致性、区分度、剖面高度 |
| RIASEC 相邻/相对 | `HEX-adjacent` / `HEX-opposite` | 识别内在一致性与张力 |
| CliftonStrengths Top 5 | `STR-<theme>` × 5 | 主题行为定义、领域归属、组合叙事 |
| CliftonStrengths 主导领域 | `DOMAIN-<name>` | 工作方式偏好、团队角色 |
| 整合结果 | `AFF-<职业集群>` | 兴趣-优势共同指向的候选方向 |
| 潜在张力 | `TENS-<description>` | 需要重点讨论的职业议题 |

### 6.3 兴趣-优势亲和度估算（专业版）

> 注意：以下算法为启发式估算，未经验证，仅用于结构化讨论。

**兴趣匹配度（Interest Match）**：

| 匹配情况 | 得分 | 说明 |
|---------|------|------|
| 学生 RIASEC 第一码 = 集群主类型 | +3 | 强兴趣匹配 |
| 学生 RIASEC 第二码 = 集群第二类型 | +2 | 中等兴趣匹配 |
| 学生 RIASEC 第三码 = 集群第三类型 | +1 | 弱兴趣匹配 |
| 学生 RIASEC 类型与集群主类型相邻 | +1 | 相容补充 |

**优势匹配度（Strength Match）**：

| 匹配情况 | 得分 | 说明 |
|---------|------|------|
| Top 5 主题出现在集群 `STR-` 列表中 | 每个 +2 | 强优势匹配 |
| Top 6-10 主题出现在集群 `STR-` 列表中 | 每个 +1 | 中等优势匹配 |
| 第一主题匹配 | 额外 +1 | 高能量匹配 |

**综合亲和度 = 兴趣匹配度 + 优势匹配度**

### 6.4 专业版输出：候选人报告（节选示例）

**学生 D：RIASEC = I-A-S；Top 5 = Learner, Analytical, Ideation, Relator, Communication**

| 候选职业集群 | 兴趣匹配 | 优势匹配 | 综合亲和度 | 可能性描述 | 潜在张力 |
|------------|---------|---------|-----------|-----------|---------|
| 科学传播者 / 科普作家 | I(3)+A(2)+S相邻(1)=6 | Learner(2)+Analytical(2)+Ideation(2)+Communication(2)+第一主题(1)=9 | **15** | 兴趣（I+A）与优势（研究+创意+表达）高度一致 | 学术严谨性与大众可读性之间的平衡 |
| 用户研究员 | I(3)+A(2)+S(1)=6 | Learner(2)+Analytical(2)+Relator(2)+第一主题(1)=7 | **13** | I 与 Strategic Thinking 优势明显，Relator 支持深度访谈 | 定量数据与定性洞察的整合 |
| 教育心理学家 | I(2)+S(3)+A(1)=6 | Learner(2)+Relator(2)+Communication(1)=5 | **11** | S 兴趣与 Relator 优势匹配 | 需要长期学术训练与情感边界管理 |
| 数据分析师 | I(3)+C相邻(0)=3 | Analytical(2)+Learner(2)=4 | **7** | 优势匹配，但 A 兴趣可能得不到满足 | 创意表达空间不足 |

### 6.5 专业版对话提示

| 议题 | 提问示例 |
|-----|---------|
| 兴趣-优势一致 | “你的 RIASEC 显示你对研究型工作有高兴趣，同时你的 Top 5 中 Analytical 和 Learner 靠前。这两者如何共同说明你可能的学习/工作方式？” |
| 兴趣-优势张力 | “你对艺术型（A）工作有兴趣，但你的优势主题更偏向执行和结构化。这对你把创意落地可能意味着什么？” |
| 环境偏好 | “你的 Consistency 和 Discipline 靠前，说明你喜欢有序环境；但你的 Enterprising 兴趣指向高变化环境。你如何在变化中创造秩序？” |
| 社会互动 | “你的 Empathy 和 Relator 支持深度关系，但 Woo 不在 Top 5。这是否意味着你更适合一对一而非大规模社交？” |
| 发展路径 | “如果要在 3 年后成为科学传播者，你现在可以补充哪些技能、经验和人脉？” |

### 6.6 专业版伦理检查清单

- [ ] 是否向学生说明测评目的、局限性与保密性？
- [ ] 是否避免使用“适合/不适合”等绝对化语言？
- [ ] 是否区分了兴趣、优势与能力？
- [ ] 是否考虑了学生的文化背景、家庭期望与社会经济因素？
- [ ] 是否将结果作为“当前快照”而非“命运标签”？
- [ ] 是否鼓励学生通过实践验证而非仅依赖测评？
- [ ] 是否在报告中标注了证据等级（RIASEC [A-B]，CliftonStrengths [B-C]，整合 [C]）？

| **目标** | 快速自我了解，激发探索 | 结构化解读，支持深度咨询 |
| **语言** | 比喻化、标签化、行动导向 | 术语化、证据化、分析导向 |
| **输出** | 一页纸报告 + 3 个探索方向 | 完整报告 + 亲和度矩阵 + 对话提示 |
| **矩阵使用** | 找到自己的交叉象限即可 | 交叉象限 + 主次类型 + 主题组合分析 |
| **Tag 深度** | 学生版 Tag（如 #助人连接派） | 专业 Tag（如 `RIASEC-S-A-E`、`STR-empathy`） |
| **张力处理** | 简单提醒“注意兴趣≠能力” | 系统分析兴趣-优势张力、环境-优势张力 |
| **证据等级** | 不需要呈现 | 必须标注 `[A]` `[B]` `[C]` `[D]` |
| **伦理边界** | 强调“测试不是判决书” | 签署知情同意、保护隐私、避免高利害使用 |

## 10. 局限性与使用边界

### 10.1 主要局限

1. **理论整合未验证**：RIASEC 与 CliftonStrengths 的整合基于 Big Five 理论桥梁与专家判断，未经过大规模实证研究。
2. **文化适用性**：CliftonStrengths 在美国组织文化中开发，中文语境下的效度研究有限；RIASEC 中文版本的结构效度也存在争议。
3. **亲和度算法为启发式**：第 6.3 节的评分方法仅用于结构化讨论，不可作为科学预测依据。
4. **职业快速变化**：矩阵中的职业示例需要随劳动力市场变化定期更新。

### 10.2 使用边界

- 不用于选拔、分班、评奖等 high-stakes 决策。
- 不替代专业心理诊断或临床评估。
- 不忽视学生的能力、价值观、家庭背景与现实条件。
- 应由受过培训的人员进行专业版解读。

---

## 11. 结语

本框架的核心目标不是告诉学生“你应该做什么”，而是帮助他们：

1. **分开理解**自己的兴趣与优势；
2. **用 Tag 和矩阵**结构化地看到两者之间的关系；
3. **在真实行动中**验证、调整、成长。

> **学生版的终点是“开始探索”，专业版的终点是“促进一场更好的职业对话”。**
