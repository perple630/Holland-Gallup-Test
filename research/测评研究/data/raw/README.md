## 2. CliftonStrengths 报告结构 / CliftonStrengths Report Structure

### 2.1 报告类型
CliftonStrengths 主要提供两类报告：

1. **Signature Themes Report（Top 5 报告）**：列出得分最高的 5 个主题，含主题描述与“行动建议（Ideas for Action）”。
2. **CliftonStrengths 34 Report**：列出全部 34 个主题的排序，并分为四大领域（domains）：
   - **Executing（执行力）**
   - **Influencing（影响力）**
   - **Relationship Building（关系建立）**
   - **Strategic Thinking（战略思维）**

### 2.2 报告常见内容模块

| 模块 | 说明 |
|------|------|
| 主题排序 / Theme Sequence | 34 个主题从第 1 到第 34 的完整排序 |
| 领域分布 / Domain Summary | 各领域中主题数量与能量分布 |
| 主题描述 / Theme Descriptions | 每个主题的标准化定义与行为表现 |
| 个性化洞察 / Strengths Insight | 基于主题组合生成的个性化描述（Top 5 报告常见） |
| 行动建议 / Ideas for Action | 每个主题 10 条左右的发展建议 |
| 主题对比 / Theme Contrast | 相邻主题或高-低主题之间的差异说明 |

### 2.3 样例报告中的 Top 5 示例

在公开样例中常见的 Top 5 组合包括：
- Achiever → Relator → Developer → Includer → Positivity
- Achiever → Learner → Responsibility → Focus → Strategic

（来源：`cliftonstrengths-34-sample.pdf`）

## 3. Self-Directed Search（SDS）报告结构 / SDS Report Structure

### 3.1 报告类型
SDS 提供多种报告形式：

1. **You and Your Career**：7 页可重复使用的纸质说明手册。
2. **Occupations Finder (OF)**：包含 1,334 种职业及其 Holland 编码的职业查询手册。
3. **Educational Opportunities Finder (EOF)**：按 Holland 编码分类的 750+ 学习领域查询手册。
4. **SDS Interpretive Report (SDS:IR)**：10–12 页的计算机生成个性化报告，含职业、学习领域与休闲活动建议。
5. **Professional Interpretive Report**：为咨询师提供的专业版报告，含 7 项诊断指标。

### 3.2 SDS 解读的 7 项诊断指标

| 指标 | 英文 | 说明 |
|------|------|------|
| 一致性 / Congruence | Congruence | 个人 Holland 代码与目标职业代码的匹配程度 |
| 总结代码 / Summary Code | Summary Code | 三字母 Holland 代码（如 SAE、RIA） |
| 职业梦想一致性 / Coherence of Aspirations | Coherence of Aspirations | 列出的职业梦想之间是否一致 |
| 内部一致性 / Consistency | Consistency | 前三码中前两个字母在六边形上的邻近程度 |
| 剖面高度 / Profile Elevation | Profile Elevation | 六个量表得分总和，反映参与程度 |
| 区分性 / Differentiation | Differentiation | 剖面是否清晰（高分与低分差距） |
| 常见性 / Commonness | Commonness | 该三码在人群中的出现频率 |

（来源：PAR, Inc. SDS Professional Interpretive Report 样例）

### 3.3 样例报告中的代码示例

公开样例中显示一例 SAE 代码：

| RIASEC 类型 | 得分 |
|------------|------|
| Realistic (R) | 25 |
| Investigative (I) | 24 |
| Artistic (A) | 31 |
| Social (S) | 37 |
| Enterprising (E) | 27 |
| Conventional (C) | 21 |

**Summary Code: SAE**（Social-Artistic-Enterprising）

（来源：PAR, Inc. SDS Client Interpretive Report 样例）

## 4. 实践体系 / Practice Systems

### 4.1 CliftonStrengths 教练框架

盖洛普认证的教练框架通常包括以下核心对话：

| 对话阶段 | 目的 |
|---------|------|
| 建立关系 / Establish Relationship | 建立信任与教练同盟 |
| 理解优势剖面 / Understand Profile | 解读 Top 5 或 34 主题报告 |
| 欣赏才能 / Appreciate Talents | 加深对个人自然模式的认识 |
| 投资才能 / Invest in Talents | 将才能转化为可应用的优势 |

常见实践模型：
- **Name It, Claim It, Aim It**：识别（Name）→ 接纳（Claim）→ 定向应用（Aim）。
- **团队教练对话**：从个人优势 → 伙伴关系 → 团队 DNA → 集体最佳表现。

（来源：Gallup Accelerated Strengths Coaching Course 资料）

### 4.2 SDS/Holland 职业咨询框架

SDS 通常与以下咨询流程结合：

| 阶段 | 活动 |
|------|------|
| 测评施测 | 完成 SDS 四个分量表 |
| 代码解读 | 确定三字母 Holland 代码 |
| 职业探索 | 使用 Occupations Finder 查找匹配职业 |
| 教育规划 | 使用 Educational Opportunities Finder 查找学习领域 |
| 一致性分析 | 评估个人代码与目标职业/专业的一致性 |
| 决策支持 | 结合 CASVE 循环等职业决策模型 |

（来源：PAR, Inc. SDS Handbook; Niles & Harris-Bowlsbey, 2017）

### 4.3 高校应用示例

- **CliftonStrengths**：常用于新生适应、领导力项目、简历与面试辅导（如 Luther College Strengths Guide）。
- **SDS / RIASEC**：常用于职业中心的专业选择咨询、职业探索工作坊。

## 6. 局限性与使用注意 / Limitations and Usage Notes

1. **版权与访问限制**：部分官方样例报告由测评出版商提供，直接下载可能受策略限制；本汇编仅记录公开可访问的 URL。
2. **样例不代表完整报告**：样例报告通常为演示目的生成，可能省略了部分付费版内容。
3. **不可用于正式测评**：本汇编仅供研究参考，不可替代正式测评与专业咨询。
4. **文化适用性**：英文样例报告基于英语语境，中文版报告的内容与解释可能存在差异。
