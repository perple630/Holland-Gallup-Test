# 盖洛普-霍兰德职业测评整合研究项目
# Gallup–Holland Career Assessment Integration Research Project

本项目致力于以**心理测量学（psychometrics）**的严格标准，系统研究 Gallup CliftonStrengths 与 Holland RIASEC 两类职业测评的**信度（reliability）**、**效度（validity）**与**实用性（utility）**，并通过 TAG 标注系统探索两者在学生学业选择、专业选择与职业发展中的整合应用。

## 项目目标 / Project Goals

1. 系统梳理两类测评的研究文献与信效度证据；
2. 收集并整理测评结果样例与实践体系；
3. 评估 Holland 职业集群分类的合理性与当代适用性；
4. 设计并实施 TAG 标注系统，桥接 Gallup 优势主题与 Holland 兴趣类型；
5. 设计多种整合思路，并通过 TAG 数据验证其理论合理性与潜在信效度；
6. 构建双维度的学生学业与职业发展综合评价体系；
7. 开发研究成果展示网站。
