# 证据分级细则 / Evidence Grading Protocol

## 目的 / Purpose

为了在文献综述与结论推导中保持学术严谨性，本项目对引用证据实行 A–D 四级分级。所有经验性结论均应标注证据等级，避免将低质量证据误作为强证据使用。

## 分级标准 / Grading Criteria

### Level A — 强证据 / Strong Evidence

**标准**：
- 多项独立大样本研究（N ≥ 500）；
- 使用现代测量方法，如验证性因素分析（CFA）、结构方程模型（SEM）、项目反应理论（IRT）；
- 在不同文化或语言群体中进行了复制或跨文化验证；
- 发表于同行评审期刊，方法透明，数据可获取或可请求。

**适用场景**：可作为核心结论的主要支撑。

### Level B — 中等证据 / Moderate Evidence

**标准**：
- 一项或多项中等样本研究（N ≈ 100–499）；
- 方法学基本合理，可能存在一定局限（如横断面设计、单一来源数据）；
- 有独立研究者或机构报告支持；
- 未在不同群体中充分复制。

**适用场景**：可作为初步结论或假设的支持，但需谨慎推广。

### Level C — 弱证据 / Weak Evidence

**标准**：
- 小样本研究（N < 100）；
- 方法学存在明显缺陷（如缺乏对照组、测量工具未充分验证、仅描述性统计）；
- 回顾性研究或案例系列。

**适用场景**：仅用于提示可能方向，不可作为结论性证据。

### Level D — 非证据 / Non-Evidence

**标准**：
- 测评机构宣传材料、营销文案、白皮书；
- 专家个人意见、博客、未经验证的案例报告；
- 无法追溯原始数据或研究设计的二手转述。

**适用场景**：可用于说明工具的市场定位、常见使用方式或用户叙述，但不可作为科学有效性的证据。

## 使用规则 / Usage Rules

1. 每个经验性陈述后应尽量标注证据等级，如：`(Author, Year) [B]`。
2. 若同一结论同时有 A 与 B 级证据，优先引用 A 级。
3. Level D 材料不可单独支撑任何科学结论。
4. 在综合评述中，应明确说明证据基础的总体强度与主要局限。

## 降级与升级条件 / Upgrade / Downgrade Conditions

- **可升级**：当独立研究复制了原有发现，或使用了更严谨的测量方法时，可考虑升级。
- **应降级**：当发现方法学缺陷、样本偏差、不可重复或存在利益冲突时，应降级。

## 批判性反思 / Critical Reflection

证据分级本身并非绝对。研究者应始终考虑：
- 发表偏倚（publication bias）；
- 文件抽屉效应（file drawer effect）；
- 利益冲突（conflicts of interest）；
- 测量工具的版本差异（如 CliftonStrengths 各版本、SDS 不同语言修订版）。
